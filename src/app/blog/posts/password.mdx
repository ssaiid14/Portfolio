## Añadir páginas protegidas

Para habilitar la protección con contraseña para páginas específicas, añade las rutas de las páginas que quieres proteger al objeto `protectedRoutes` en el archivo `resources/config.js`.

El componente `RouteGuard` manejará automáticamente el control de acceso para estas páginas, requiriendo autenticación con contraseña antes de permitir el acceso.

<CodeBlock
    marginBottom="16"
    highlight="2"
    codes={[
  {
    code:
`const protectedRoutes = {
  '/work/once-ui': true
};`,
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

El código anterior asegurará que la página `/work/once-ui` solo sea accesible después de proporcionar una contraseña.

## Establecer una contraseña

La contraseña se puede establecer en el archivo `.env.local` para desarrollo local y como [variable de entorno](https://vercel.com/docs/environment-variables) para producción.
