## Gestionar proyectos

Puedes crear, editar y eliminar proyectos añadiendo, modificando o eliminando archivos `*.mdx` en el directorio `src/app/work/projects`.

## Frontmatter

El frontmatter se usa para establecer los metadatos del proyecto: título, descripción, imagen y etiquetas. Se usa en la página del proyecto y en las etiquetas meta y open graph.

## Sintaxis MDX

Puedes usar MDX para escribir el contenido de tus proyectos. Te ayuda a escribir contenido rico y dinámico con código mínimo. Algunos elementos MDX se transformarán automáticamente a componentes de Once UI para integrarse mejor en el diseño y añadir funcionalidad adicional.

## Componentes personalizados

Puedes usar componentes personalizados en archivos MDX, pero necesitas importarlos primero en el archivo `src/components/mdx.tsx`.

<CodeBlock
    marginBottom="16"
    highlight="11-12"

        label: "src/components/mdx.tsx"

Como puedes ver, los componentes `Table` y `CodeBlock` ya están importados y disponibles para usar. Puedes añadir más simplemente importándolos en este archivo y pasándolos al objeto `components`.
