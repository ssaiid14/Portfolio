## Código

Usa el componente CodeBlock para mostrar fragmentos de código con resaltado de sintaxis, vista previa y funcionalidad de copiar al portapapeles. Usa el componente InlineCode para fragmentos de código en línea.

### CodeBlock

El componente code block funciona basándose en el CodeBlock de Once UI. Puedes acceder a él con la sintaxis completa para personalización total, o usando la sintaxis estándar de markdown (triple comillas invertidas) con props preconfiguradas.

<Column fillWidth gap="-1" marginBottom="16" marginTop="8">
<CodeBlock
    radius="none"
    topRadius="l"
    codes={[
      {
        code:
`// Sintaxis completa del componente
<CodeBlock
    codes={[
      {
        code: "console.log('¡Hola, Mundo!');",
        language: "javascript",
        label: "Ejemplo"
      }
    ]}
/>

// Ejemplo de sintaxis corta
\`\`\`tsx
function greeting(name) {
    return \`¡Hola, \${name}!\`;
}

console.log(greeting('Mundo'));
\`\`\``,
        language: "tsx",
        label: "CodeBlock"
      }
    ]}
/>
<Row fillWidth bottomRadius="l" padding="4" border="neutral-medium" background="surface">
    <Button href="https://once-ui.com/docs/codeBlock" weight="default" label="Ver en Once UI" size="s" variant="tertiary" suffixIcon="chevronRight"/>
</Row>
</Column>

### InlineCode

El `InlineCode` es otro componente de Once UI al que puedes acceder ya sea a través de la sintaxis completa o la sintaxis corta (comillas invertidas). Generalmente se recomienda la sintaxis estándar de markdown, ya que no tiene tantas opciones de configuración.

<Column fillWidth gap="-1" marginBottom="16" marginTop="8">
<CodeBlock
    radius="none"
    topRadius="l"
    codeHeight={24}
    preview={
    <InlineCode>
        const x = 10;
    </InlineCode>}
    codes={[
      {
        code:
`// Sintaxis completa del componente
<InlineCode>
    const x = 10;
</InlineCode>

// Ejemplo de sintaxis corta
\`const x = 10;\``,
        language: "tsx",
        label: "InlineCode"
      }
    ]}
/>
<Row fillWidth bottomRadius="l" padding="4" border="neutral-medium" background="surface">
    <Button href="https://once-ui.com/docs/inlineCode" weight="default" label="Ver en Once UI" size="s" variant="tertiary" suffixIcon="chevronRight"/>
</Row>
</Column>

## Componentes personalizados

Como Magic Portfolio está basado en Once UI, puedes usar todos los componentes proporcionados por Once UI en los archivos `.mdx`, pero necesitas importarlos primero en el archivo `src/product/mdx.tsx` y pasarlos al objeto `components`.

<CodeBlock
    marginBottom="16" marginTop="8"
    highlight="2"
    codes={[
      {
        code:
`const components = {
    // Añadir nuevos componentes
    p: createParagraph as any,
    h1: createHeading("h1") as any,
    ...
    Media,
    SmartLink,
};`,
        language: "tsx",
        label: "src/product/mdx.tsx"
      }
    ]}
/>
