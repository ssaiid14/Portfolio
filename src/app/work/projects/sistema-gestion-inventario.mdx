## Descripción

Sistema completo de gestión de inventario que incluye un dashboard administrativo moderno y una API backend robusta construida con Clean Architecture. El proyecto implementa funcionalidades avanzadas como sistema de tags, timeline de auditoría, comentarios y layouts personalizables.

## Tecnologías Utilizadas

### Backend
- **TypeScript** - Tipado estático
- **Fastify** - Framework web rápido
- **Prisma** - ORM moderno
- **PostgreSQL** - Base de datos relacional
- **Zod** - Validación de esquemas

### Frontend
- **React 18** - Biblioteca de UI
- **Vite** - Build tool moderno
- **Tailwind CSS** - Framework de CSS
- **Framer Motion** - Animaciones
- **Radix UI** - Componentes accesibles

## Características Principales

- Autenticación JWT con refresh tokens
- Sistema de tags para categorización flexible
- Timeline de auditoría para seguimiento de cambios
- Dashboard layouts personalizables por usuario
- Gestión completa de productos, ventas y clientes
- Sistema de comentarios con soporte de hilos
- Rate limiting y seguridad robusta
- Testing completo con Vitest

## Arquitectura

El backend implementa Clean Architecture con capas bien definidas: domain, application, infrastructure y presentation. El frontend utiliza componentes reutilizables con estado centralizado.
