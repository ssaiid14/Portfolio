## Descripción

Sistema completo de gestión de inventario desarrollado como proyecto del Grado Superior en Desarrollo de Aplicaciones Web. La aplicación permite gestionar productos, categorías, proveedores y generar reportes.

## Tecnologías Utilizadas

- **PHP** - Backend y lógica de negocio
- **MySQL** - Base de datos relacional
- **Bootstrap** - Framework CSS
- **JavaScript** - Interactividad frontend

## Funcionalidades

- CRUD completo de productos y categorías
- Sistema de autenticación de usuarios
- Roles y permisos (admin/usuario)
- Reportes de inventario en PDF
- Panel de administración
- Búsqueda y filtrado avanzado

## Aprendizajes

Este proyecto me permitió profundizar en:
- Arquitectura MVC
- Seguridad en aplicaciones web
- Diseño de bases de datos relacionales
- Gestión de sesiones y autenticación
