## Descripción

Portfolio profesional desarrollado desde cero utilizando las últimas tecnologías web. El proyecto demuestra mis habilidades en desarrollo frontend moderno y diseño responsivo.

## Tecnologías Utilizadas

- **Astro 5** - Framework de generación estática
- **React 18** - Componentes interactivos
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Estilos utilitarios
- **GSAP** - Animaciones avanzadas

## Características Principales

- Diseño responsivo optimizado para todos los dispositivos
- Modo oscuro/claro con persistencia
- Formulario de contacto funcional con EmailJS
- Animaciones fluidas y profesionales
- Optimización SEO completa
- Rendimiento optimizado (Lighthouse 95+)
