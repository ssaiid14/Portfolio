# Mi Portfolio

Portfolio personal construido con [Next.js](https://nextjs.org) y [Once UI](https://once-ui.com).

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Configuración

- `src/resources/once-ui.config.ts` - Configuración del tema
- `src/resources/content.js` - Contenido del portfolio

## Basado en

[Magic Portfolio](https://github.com/once-ui-system/magic-portfolio) por Once UI
